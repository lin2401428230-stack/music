# 音乐播放器 MCP 插件

## 描述
这是一个 MCP 音乐播放器插件，支持导入音乐文件，AI 可以通过自然语言自由控制播放、暂停、恢复和停止，并支持后台播放。

## 触发词
- 播放音乐
- 暂停音乐
- 恢复播放
- 停止音乐
- 音乐状态
- 列出音乐
- 播放 [歌曲名]

## 工具 (Tools)

### 1. list_songs
**功能**：列出所有已导入的音乐文件。
**用法**：用户询问“有哪些音乐”或“列出所有歌曲”时触发。

### 2. play_song
**功能**：播放指定名称的音乐，支持后台播放。
**参数**：
- `name` (string, 必填): 音乐文件名。
**用法**：用户说“播放 [歌曲名]”时触发。

### 3. pause_music
**功能**：暂停当前正在播放的音乐。
**用法**：用户说“暂停音乐”时触发。

### 4. resume_music
**功能**：恢复播放已暂停的音乐。
**用法**：用户说“继续播放”或“恢复音乐”时触发。

### 5. stop_music
**功能**：停止播放音乐。
**用法**：用户说“停止音乐”时触发。

### 6. music_status
**功能**：查询当前音乐播放状态。
**用法**：用户说“现在在放什么歌”或“音乐状态”时触发。

## 注意事项
- 播放音乐前，请确保已通过界面或后台导入音乐文件。
- 所有播放控制均支持后台运行，关闭界面不影响音乐播放。
